About the Project

Stylize your text with fonts!

Hi, I'm Arif Faisal. I am an aspiring Full Stack Developer.

Often, it has seemed to me that, it gets quite confusing while choosing the right font for your project or content. Yes, we do have MS Word to try it out. But, wouldn't it be simple where we can generate some random texts, and try out different fonts to see which would suit our content.

This project is a fruition to that. The project aims to add more fonts periodically so that the user can try it out, and choose the right one for themselves.
